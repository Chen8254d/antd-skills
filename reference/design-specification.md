# Ant Design 设计规范

供设计师在 Figma 中创建符合规范的设计稿。

## 🎨 颜色规范

### 品牌色

| 名称 | 色值 | 用途 |
|------|------|------|
| Primary | `#1677ff` | 主操作、链接、选中态 |
| Primary Hover | `#4096ff` | 主色悬停态 |
| Primary Active | `#0958d9` | 主色按下态 |

### 功能色

| 名称 | 色值 | 用途 |
|------|------|------|
| Success | `#52c41a` | 成功状态 |
| Warning | `#faad14` | 警告状态 |
| Error | `#ff4d4f` | 错误状态 |
| Info | `#1677ff` | 信息提示 |

### 中性色

| 名称 | 色值 | 用途 |
|------|------|------|
| Title | `rgba(0,0,0,0.88)` | 标题文字 |
| Text | `rgba(0,0,0,0.88)` | 正文文字 |
| Secondary | `rgba(0,0,0,0.65)` | 次要文字 |
| Disabled | `rgba(0,0,0,0.25)` | 禁用文字 |
| Border | `#d9d9d9` | 边框 |
| Divider | `rgba(5,5,5,0.06)` | 分割线 |
| Background | `#ffffff` | 容器背景 |
| Layout BG | `#f5f5f5` | 布局背景 |

## 📐 间距规范

使用 **4px** 为基础单位的间距系统：

| 级别 | 数值 | 用途 |
|------|------|------|
| XXS | 4px | 紧凑间距 |
| XS | 8px | 小间距 |
| SM | 12px | 中小间距 |
| MD | 16px | 标准间距 |
| LG | 24px | 大间距 |
| XL | 32px | 超大间距 |
| XXL | 48px | 区块间距 |

### 常用间距场景

- 按钮内边距：`12px 16px`
- 卡片内边距：`24px`
- 表单项间距：`24px`（垂直）
- 列表项间距：`16px`
- 页面边距：`24px`

### 字体家族

```
-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, 'Helvetica Neue', Arial, 'Noto Sans', sans-serif
```

### 字号

| 级别 | 字号 | 行高 | 用途 |
|------|------|------|------|
| H1 | 38px | 46px | 一级标题 |
| H2 | 30px | 38px | 二级标题 |
| H3 | 24px | 32px | 三级标题 |
| H4 | 20px | 28px | 四级标题 |
| H5 | 16px | 24px | 五级标题 |
| Body | 14px | 22px | 正文 |
| Small | 12px | 20px | 辅助文字 |

### 字重

- Regular: 400（正文）
- Medium: 500（小标题）
- Semibold: 600（标题）

## 📦 组件规范

### Button 按钮

| 尺寸 | 高度 | 内边距 | 字号 |
|------|------|--------|------|
| Large | 40px | 16px 24px | 16px |
| Middle | 32px | 12px 16px | 14px |
| Small | 24px | 8px 12px | 14px |

**状态样式**：
- Default: 白底 + 边框 `#d9d9d9`
- Primary: 主色填充 `#1677ff`
- Danger: 红色填充 `#ff4d4f`
- Disabled: 灰色背景 `#f5f5f5` + 文字 `rgba(0,0,0,0.25)`

### Input 输入框

| 尺寸 | 高度 | 内边距 |
|------|------|--------|
| Large | 40px | 12px |
| Middle | 32px | 8px 12px |
| Small | 24px | 4px 8px |

**状态**：
- Normal: 边框 `#d9d9d9`
- Hover: 边框 `#4096ff`
- Focus: 边框 `#1677ff` + 外发光
- Error: 边框 `#ff4d4f`
- Disabled: 背景 `#f5f5f5`

### Card 卡片

- 背景：`#ffffff`
- 圆角：8px
- 内边距：24px
- 阴影：`0 1px 2px 0 rgba(0,0,0,0.03), 0 1px 6px -1px rgba(0,0,0,0.02), 0 2px 4px 0 rgba(0,0,0,0.02)`

### Table 表格

- 表头背景：`#fafafa`
- 表头字重：600
- 行高：54px
- 边框：`#f0f0f0`
- 斑马纹：`#fafafa`

### Modal 弹窗

- 宽度：520px（默认）
- 圆角：8px
- 标题字号：16px
- 内边距：24px
- 遮罩：`rgba(0,0,0,0.45)`

## 📱 栅格系统

基于 **24 栏** 栅格：

| 断点 | 屏幕宽度 | 栏宽 |
|------|----------|------|
| xs | < 576px | 自动 |
| sm | ≥ 576px | 自动 |
| md | ≥ 768px | 自动 |
| lg | ≥ 992px | 自动 |
| xl | ≥ 1200px | 自动 |
| xxl | ≥ 1600px | 自动 |

栏间距：16px（推荐）

---

## 🖼️ 图标规范

- 图标尺寸：16px / 20px / 24px
- 图标颜色：跟随文字颜色
- 线宽：2px
- 图标与文字间距：8px

推荐使用 Ant Design Icons：
https://ant.design/components/icon-cn

## ✅ 设计检查清单

在提交设计稿前，请检查：

- [ ] 颜色使用规范中定义的色值
- [ ] 间距为 4 的倍数
- [ ] 字号使用规范中的尺寸
- [ ] 圆角使用规范中的值
- [ ] 组件状态完整（Normal/Hover/Active/Focus/Disabled）
- [ ] 响应式适配（至少适配 1440px 和 1920px）
- [ ] 图标使用 Ant Design Icons
- [ ] 布局符合栅格系统
